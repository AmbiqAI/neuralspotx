# neuralspotx prebuilt metadata contract

Metadata manifests define ABI/compatibility fields only:

- `NSX_PREBUILT_SOC_FAMILY`
- `NSX_PREBUILT_TOOLCHAIN`
- `NSX_PREBUILT_ABI_FLAGS`
- `NSX_PREBUILT_RELEASE`
- `NSX_PREBUILT_AMBIQSUITE_VERSION`
- `NSX_PREBUILT_MODULE_API_VERSION`

Library locations are configured with:

- `NSX_HAL_LIB`
- `NSX_BSP_LIB`

Path-bearing manifests are intentionally not used.
