# apollo510dL_evb

Built-in NSX board definition for the Apollo510L (`apollo510dL_evb`) target.

The Apollo510L is part of the Apollo330P / Apollo510L family (Cortex-M55,
thumbv8.1m-fpv5-hard, J-Link device `Apollo330P_510L`). It is served by the
`nsx-ambiqsuite` SDK provider — prebuilt HAL and BSP libraries live under
`sdk/lib/apollo510L/`.
