arena is either **cold** (the arena buffer is the read-only blob
in source memory; kernels read it in place) or **staged** (the
arena buffer is a writable runtime copy hydrated from a separate

## 🛠️ Integration Guide

The module ships an `nsx-module.yaml` and `CMakeLists.txt` for the neuralspotx (NSX) build system, exposing `nsx::resnet_aot` as a CMake target.

### **STEP 1:** Add **nsx-cmsis-nn** dependency to your project

Ensure the **nsx-cmsis-nn** module is available in your NSX modules directory:

```bash
cp -r <path-to-nsx-cmsis-nn> <project-root>/modules/nsx-cmsis-nn
```

---

### **STEP 2:** Add resnet-aot module to your project

Copy the **resnet-aot** module into your project **modules** directory:

```bash
cp -r <path-to-resnet-aot> <project-root>/modules/resnet-aot
```

Alternatively, you can have **heliaAOT** generate directly into your project modules directory:

```bash
helia-aot convert ... --module.type nsx --module.path <project-root>/modules
```

---

### **STEP 3:** Update your application CMakeLists.txt

In your app CMakeLists.txt:

```cmake
add_subdirectory(modules/nsx-cmsis-nn)
add_subdirectory(modules/resnet-aot)

add_executable(app
  src/main.c
  # ... your sources ...
)

target_link_libraries(app PRIVATE
  nsx::cmsis_nn
  nsx::resnet_aot
)
```

> The module links `nsx::cmsis_nn` and `${NSX_BOARD_FLAGS_TARGET}` automatically.

---

### **STEP 4:** *(Optional)* Override memory placement

To place model tensors/arenas in specific memory sections, create an attributes header and pass it at configure time:

```cmake
set(RESNET_AOT_ATTRIBUTES_HEADER
    "${CMAKE_CURRENT_SOURCE_DIR}/my_attributes.h")
```

This header will be force-preincluded, overriding the default no-op macros in `resnet_platform.h`.

## 🚀 Usage Example

Minimal runtime sequence in your `main.c`:

```c
#include "resnet_model.h"
#include "resnet_test_case.h"  // optional

// Define model context and callback
static void resnet_model_operator_cb(
    int32_t op,
    resnet_operator_state_t state,
    int32_t status,
    void *user_data
) {
    switch (state) {
        case resnet_op_state_init_finished:
            AOT_PRINTF("Operator %d init finished with status %d\n", op, status);
            break;
        case resnet_op_state_run_finished:
            AOT_PRINTF("Operator %d run finished with status %d\n", op, status);
            break;
    }
}

resnet_model_context_t resnet_model_ctx = {
    // Optional callback- useful for debug/profiling/context
    .callback = resnet_model_operator_cb,
    .user_data = NULL
};

int main(void) {

    int32_t status;

    // 1. Initialize model
    //    model_init calls context_init, hydrates any staged-constant
    //    arenas (via the weak resnet_hydrate_constants — override
    //    this symbol to drive hydration with DMA or any other
    //    mechanism), and then runs every operator's _init hook.
    status = resnet_model_init(&resnet_model_ctx);

    // 2. Copy input tensors to context (zero for this example)
    for (int i = 0; i < resnet_num_inputs; i++) {
         arm_memset_s8(
            resnet_model_ctx.inputs[i].data,
            0,
            resnet_model_ctx.inputs[i].size
        );
    }

    // 3. Invoke model
    status = resnet_model_run(&resnet_model_ctx);

    // 4. Retrieve output tensors from context
    // ...

    // Alternatively, run included smoke test
    resnet_test_case_init();
    status = resnet_test_case_run();

    return 0;
}
```
